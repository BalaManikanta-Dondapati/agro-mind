# 🌿 AgroMind: Crop Mood Estimation

AI-based system to analyze crop health using **NDVI (satellite-inspired features) + Farmer Sentiment Analysis**

---

## 🚀 Features

* 📷 Image-based crop health prediction (Healthy / Moderate / Stressed)
* 💬 Farmer sentiment analysis using NLP
* 🔗 Feature fusion (NDVI + Sentiment)
* 📊 Interactive visual insights (confidence, NDVI trends)
* 🧠 Explainable AI (feature importance)

---

## 🛠️ Tech Stack

* Python
* Streamlit
* Scikit-learn
* Pandas / NumPy
* OpenCV
